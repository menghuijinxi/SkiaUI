# Agent Instructions

本项目的 C++ 代码必须遵守 [C++ 代码规范](docs/cpp-code-standard.md)。

## 对 AI 的硬性要求

1. 修改或生成 C++ 代码前，先阅读 `docs/cpp-code-standard.md`。
2. 用户在当前对话里的明确要求优先级最高；其次是本文件；然后是 `docs/cpp-code-standard.md`；最后才是现有代码习惯。
3. 如果规范与现有代码冲突，优先遵守规范；除非改动范围会明显扩大，则先说明冲突并采用最小可行改动。
4. 不要为了“顺手优化”重构无关代码。除非用户明确要求，改动必须围绕当前任务。
5. 写完 C++ 代码后，尽量运行项目已有的格式化、构建和测试命令；不能运行时要说明原因。
6. 如果必须偏离规范，必须在最终回复里说明偏离点和原因。
